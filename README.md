#### 태그 성질에 따른분류
1. 블록요소 태그
    - html, body, div, p, ul, ol, li, dl, dt, dd...
    - 기본 가로 공간의 크기가 부모태그와 동일하게 인식
    - 공간값이 적용(w,h)
2. 인라인 요소 태그
    - a, span, strong, i, em ...
    - 기본 가로 공간의 크기가 안쪽에 포함된 내용만큼 인식(컨텐츠)
    - 연속해서 작성하면 가로 배열
    - 공간값이 적용 안됨
3. 인라인 블록요소 태그
    - img, input, button
    - 기본 가로 공간의 크기가 안쪽에 포함된 내용만큼 인식
    - 연속해서 작성하면 가로배열
    - 공간값이 적용된다.
